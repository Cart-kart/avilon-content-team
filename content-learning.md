## 2. WRITING STYLE & TONE

### Voice
- **Speaker:** ทีมงาน / "เอวิลอน" — writes as a team, not a CEO
- **Gender signal:** ใช้ "ค่ะ" สม่ำเสมอ (female-voiced, warm & professional)
- **Named persona:** "น้องฟ้าใส" — used as the friendly contact character (📞 098-948-9743)
- **Reader address:** ไม่ใช้ "คุณ" บ่อย — เขียนแบบรวม "ทุกท่าน", "พี่น้อง", "ผู้ที่สนใจ"

### Tone by post type
| Type | Tone |
|---|---|
| Event/Exhibition | สนุก, เชิญชวน, emoji เยอะ 🎉 |
| Partnership/MOU | ภาคภูมิใจ, เป็นทางการ, ขอบคุณเป็นพิเศษ |
| Product promotion | Benefit-led, bullet points ✅, ตัวเลขชัดเจน |
| CSR/Flood | อบอุ่น, ห่วงใย, update สด, ขอบคุณทุกคน |
| Government/CAAT | เป็นทางการมาก, ใส่ชื่อ-ตำแหน่งครบ, formal |
| International | ภาคภูมิใจ, "ก้าวสู่เวทีนานาชาติ", inspirational |

### Sentence patterns (reuse these)
- เปิดด้วย: achievement/ข่าวดี/คำถาม → อธิบาย → ขอบคุณ → CTA
- "เอวิลอน มุ่งมั่นพัฒนาเทคโนโลยีโดรน เพื่อ..."
- "ขอบคุณ [ชื่อองค์กร] ที่มอบโอกาส..."
- "นับเป็นอีกก้าวสำคัญของเรา..."
- "พร้อมให้คำปรึกษา สาธิต Demo ได้ทันที"

```

```
[หัวข้อสั้นๆ 1 บรรทัด]

[รายละเอียดงาน: วันที่ / สถานที่ / บูธ]

[CTA + เบอร์โทร "น้องฟ้าใส"]

#hashtags
```

[ขอความช่วยเหลือ/ขอบคุณ]
[ข้อมูลติดต่อ ถ้ามี]
#hashtags
```

## 10. WHAT TO AVOID

- ❌ Hard-sell / ราคา / โปรโมชัน aggressive — ไม่เคยมีในโพสต์เลย
- ❌ ตัวเลขที่ไม่มีแหล่งอ้างอิง (เช่น "ดีที่สุดในไทย" โดยไม่มี source)
- ❌ Emoji กลางประโยค Thai ที่ดูเด็กเกินไป
- ❌ ALL CAPS ภาษาไทย
- ❌ โพสต์สั้นมากโดยไม่มี context (ต่ำกว่า 3 บรรทัดมักไม่มี)
